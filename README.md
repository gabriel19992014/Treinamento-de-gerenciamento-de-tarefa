# 📝 To-Do List

Este projeto é uma aplicação simples de lista de tarefas desenvolvida em **HTML**, **CSS** e **JavaScript**.  
O objetivo é treinar e aperfeiçoar meus conhecimentos em desenvolvimento front-end, praticando conceitos como manipulação do DOM, eventos e armazenamento local com `localStorage`.

## 🚀 Funcionalidades
- Adicionar novas tarefas com título, descrição, data de vencimento e prioridade.
- Marcar tarefas como concluídas (checkbox).
- Remover tarefas individualmente.
- Diferentes cores para cada prioridade:
  - 🔴 Urgente
  - 🟡 Importante
  - 🟢 Normal
- Persistência dos dados usando **localStorage** (as tarefas continuam após recarregar a página).

## 🛠️ Tecnologias utilizadas
- **HTML5** para estrutura.
- **CSS3** para estilização e responsividade.
- **JavaScript** para lógica e interação.


## 🎯 Objetivo
Este projeto faz parte do meu processo de aprendizado e prática em front-end.  
A ideia é evoluir gradualmente, adicionando novas funcionalidades e melhorando a experiência do usuário.
